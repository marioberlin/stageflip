## Pick a surface

- **[StageFlip.Slide](/skills/stageflip/modes/stageflip-slide/)** — decks from a prompt; PPTX + PDF + video walkthrough export.
- **[StageFlip.Video](/skills/stageflip/modes/stageflip-video/)** — 30s ads across 3 aspect ratios; auto-captioned.
- **[StageFlip.Display](/skills/stageflip/modes/stageflip-display/)** — IAB/GDN HTML5 banner suites from one template.

## Under the hood

Every mode shares one engine: a deterministic RIR compiler, one tool
registry, one renderer family. The concepts, tools, runtimes, and
reference docs in the sidebar are auto-generated from the engine's
source of truth — never hand-maintained copies.

See [Concepts → RIR](/skills/stageflip/concepts/rir/) for the shared
foundation, or jump straight to the [quickstart](/quickstart/).
